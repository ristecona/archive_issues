Archive Multiple Issues

This module allows you to archive multiple Issues at once!

Very easy to use:

☛Select the Issues in list view -->

☛Action --> Archive Issues

This was not possible before

From the stock odoo instalation and modules, this feature was not there.

Make sure you are in the list view, in order to select multiple Issues.

Saves time and work.

Click on all Issues, and list view

Select multiple Issues

Click the action Archive Issues

From the wizard you can add or even remove

One click on Archive Issues and you are done!
